# Express Basics

A simple Node.js server built with Express.js — created while learning backend development basics.

## What it does

This is a minimal Express server that responds with "Hello World" when you visit the homepage.

## Tech Stack

- Node.js
- Express.js
- Nodemon (for development)

## Getting Started

### Prerequisites
- Node.js installed on your machine

### Installation

1. Clone the repository
```bash
git clone https://github.com/Aresha-Khalid072/express-basics.git
```

2. Navigate to the project folder
```bash
cd express-basics
```

3. Install dependencies
```bash
npm install
```

4. Run the server
```bash
node script.js
```

Or, for auto-restart during development:
```bash
npx nodemon script.js
```
